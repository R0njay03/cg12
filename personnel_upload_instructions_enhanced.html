Upload Process

Prepare Your Data

Download the CSV template and fill in your personnel data. Ensure all required fields are completed and follow the specified formats.

Validate Data Format

Double-check that:

  • All dates are in YYYY-MM-DD format
  • Serial numbers are unique
  • Category values are exactly "Officer" or "Non-Officer"
  • Unit codes match the accepted values

Save as CSV

Save your file in CSV format. If using Excel, choose "Save As" and select "CSV (Comma delimited)" format.

Upload File

In the Personnel Management page, click "Choose File", select your CSV file, and click "Upload & Update".

Review Results

The system will process your file and display the number of successful records and any errors encountered.

Common Issues & Solutions

Duplicate Serial Numbers

Each personnel must have a unique serial number. If you're updating existing records, the system will update the existing record with the same serial number.

Date Format Errors

All dates must be in YYYY-MM-DD format. Examples: 2023-03-15, 2020-12-01. Avoid formats like MM/DD/YYYY or DD-MM-YYYY.

Category-Specific Fields

Fill in the appropriate training fields based on the Category:

  • Officers: Use CGOC_CLASS, CGSCC_CLASS, CGSC_CLASS, etc.
  • Non-Officers: Use CGMC_CGNOC_CLASS, SPECIALIZATION, FUNCTIONAL_COURSE, etc.

Large File Uploads

For files with more than 100 records, consider splitting them into smaller batches to ensure reliable processing.

Need Help?

If you encounter issues or need assistance with the upload process:

  • Review the error messages displayed after upload
  • Check your data against the template format
  • Ensure all required fields are filled
  • Contact your system administrator for technical support
