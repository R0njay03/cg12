📋 File Format Requirements

You can upload personnel data using Excel (.xlsx, .xls) or CSV files. The file must contain the following columns in order:

Column Field Name Description Required
1 Rank Military rank (e.g., Commander, Lieutenant, Petty Officer 3) Yes
2 Last Name Personnel's last name Yes
3 First Name Personnel's first name Yes
4 Unit Code Unit identifier (e.g., CG-HQ, CG-DV, CG-SB) Yes
5 Category Must be either "Officer" or "Non-Officer" Yes
6 Remarks Additional notes or comments No

⚠️ Important Notes

File Format:
  • CSV files are recommended for best compatibility
  • Excel files require PhpSpreadsheet library (may not be available on all servers)
  • First row should contain column headers
  • Use commas to separate values in CSV files
Data Processing:
  • Duplicate records (same last name, first name, unit code) will be updated
  • Invalid categories will default to "Non-Officer"
  • Empty fields will be handled gracefully
  • Upload progress and results will be displayed

🔧 Troubleshooting

Common Issues:

  • File not uploading: Check file size and format (.xlsx, .xls, .csv only)
  • Data not importing: Verify column order and required fields
  • Excel file errors: Try converting to CSV format
  • Duplicate entries: System will update existing records automatically
